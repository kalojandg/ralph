🧪 TDD WORKFLOW
═══════════════════════════════════════════════════════════

За всеки UI таск:

  1. RED      → Write failing test
  2. GREEN    → Implement code
  3. VISUAL   → Screenshot + compare с mockup
  4. REFACTOR → Iterate until matches
  5. DONE     → Verify all + commit
